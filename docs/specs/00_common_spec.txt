# 共通基盤 詳細仕様書

## 1. 機能概要

このドキュメントは、「美容師練習管理Webアプリ V2」全体で使用される共通の要素（GASユーティリティ関数、CSSスタイル、クライアントサイドJavaScript関数、GASプロジェクト設定など）に関する仕様を定義する。これにより、アプリケーション全体の一貫性を保ち、開発効率と保守性を向上させることを目的とする。

## 2. 関連ファイル

*   `src/Utils.js`
*   `src/styles.html`
*   `src/script.html`
*   `src/appsscript.json`
*   `src/code.js` (doGet, include, 定数定義など)
*   `docs/CODING_STANDARD.txt` (コーディング規約、CSS命名規則など)

## 3. デザインコンセプト: ナチュラルモダン

*   **キーワード:** クリーン、シンプル、ナチュラル、モダン、明るい、穏やか、アクセシブル（使いやすい）
*   **雰囲気:** 白やオフホワイトを基調とした清潔感のある空間に、自然を想起させる色（グリーン系など）や洗練されたアクセントカラー（ゴールド系など）をバランス良く配置。直線と有機的な曲線を組み合わせ、モダンでありながらも温かみと使いやすさを感じさせるデザインを目指す。

## 4. 配色

*   **ベースカラー:**
    *   背景: `#f8f9fa` (Very Light Gray)
    *   コンテンツエリア背景: `#ffffff` (White)
    *   ボーダー/区切り線: `#dee2e6` (Light Gray)
    *   入力欄背景 (非活性時/読み取り専用): `#e9ecef` (Grayish White)
*   **テキストカラー:**
    *   メインテキスト: `#343a40` (Dark Gray)
    *   サブテキスト/ラベル: `#495057` (Medium Gray)
    *   リンク: `#0d6efd` (Standard Blue) - ※変更検討可
*   **アクセントカラー (プライマリ):**
    *   `#5A8F7B` (落ち着いたグリーン) - ボタン、フォーカス色、アクティブ状態など
*   **アクセントカラー (補助):**
    *   ゴールド/イエロー系: `#E4C988` (落ち着いたゴールド) - アイコン、情報表示などに少量使用
    *   ピンク系: `#FFDBC3` (淡いピーチピンク) - お知らせなど
*   **ステータスカラー:**
    *   成功: `#198754` (Green)
    *   エラー/デンジャー: `#dc3545` (Red)
    *   情報: `#0dcaf0` (Cyan)
    *   警告: `#ffc107` (Yellow)

## 5. タイポグラフィ

*   **基本フォント:** `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, Helvetica, Arial, sans-serif` (OS標準フォント優先)
*   **基本フォントサイズ:** `16px` (1rem)
*   **基本行間:** `1.6`
*   **見出し:**
    *   `h1`: `1.8rem`, `font-weight: 600`
    *   `h2`: `1.5rem`, `font-weight: 600`, `border-bottom: 1px solid #eee`
    *   `h3`: `1.25rem`, `font-weight: 600`

## 6. レイアウト方針

*   **コンテナ:**
    *   主要コンテンツの最大幅: `960px`
    *   左右パディング: `15px`
*   **主要要素間のマージン:** `1.5rem` から `2rem` 程度を基本とし、視覚的なまとまりと余白を確保する。
*   **レスポンシブ:**
    *   モバイルファーストを意識しつつ、`768px` 以下のブレークポイントでレイアウト調整（フォーム項目の縦積みなど）を行う。
    *   必要に応じて、テーブルなどの要素は横スクロールを許容する。

## 7. 共通コンポーネントスタイル方針

*   **ボタン (`.btn`):**
    *   形状: やや角丸 (`border-radius: 0.375rem`)
    *   サイズ: デフォルト (`padding: 0.6rem 1.2rem`), 小 (`.btn--small`)
    *   種類:
        *   プライマリ (`.btn--primary`): 背景 `#5A8F7B` (Primary Green)
        *   セカンダリ (`.btn--secondary`): 背景 `#6c757d` (Gray)
        *   デンジャー (`.btn--danger`): 背景 `#dc3545` (Red)
    *   ホバー/アクティブ: 背景色を少し濃くするなどの視覚的フィードバック。
    *   無効化 (`:disabled`): 透明度を下げ (`opacity: 0.65`)、クリック不可にする。
*   **フォーム入力 (`input`, `select`, `textarea`):**
    *   形状: やや角丸 (`border-radius: 0.375rem`)
    *   ボーダー: `#ced4da` (標準), フォーカス時 `#86b7fe` (Primary Greenに合わせた青系または緑系に変更検討)
    *   フォーカス時: `box-shadow: 0 0 0 0.25rem rgba(90, 143, 123, 0.25)` (Primary Greenの薄い影)
    *   読み取り専用 (`.form-display`): 背景 `#e9ecef`
    *   Select要素: 右側にシンプルな下向き矢印アイコン（SVG）を表示。
*   **メッセージ/アラート (`.message`):**
    *   形状: やや角丸 (`border-radius: 0.375rem`)
    *   表示制御: `.visible` クラスで表示、フェードイン/アウト効果。
    *   種類:
        *   成功 (`.message--success`): 背景 `#d1e7dd`, 文字 `#0f5132`
        *   エラー (`.message--error`): 背景 `#f8d7da`, 文字 `#842029`
        *   情報 (`.message--info`): 背景 `#cff4fc`, 文字 `#055160`
*   **テーブル (`.admin-table`):**
    *   ヘッダー (`thead th`): 背景 `#f8f9fa`, `font-weight: 600`
    *   行区切り線: `border-bottom: 1px solid #dee2e6`
    *   ホバー (`tbody tr:hover`): 薄い背景色 (`rgba(0, 0, 0, 0.04)`)
    *   スクロールコンテナ (`.list-container`): 縦スクロール可能、枠線付き。
*   **タブ (`.tab-btn`):**
    *   アクティブ時 (`.tab-btn.active`): 下線（Primary Green）、文字色を濃くするなど。
    *   ホバー時: 視覚的フィードバック（下線や背景色変更など）。

## 8. CSSクラス命名規則

*   **BEM (Block Element Modifier)** を採用する。
*   詳細は `docs/CODING_STANDARD.txt` を参照。

## 9. その他

*   アイコンフォントやSVGアイコンの使用は必要に応じて検討する。
*   アニメーションは控えめに、UIの応答性を高める目的で使用する。
